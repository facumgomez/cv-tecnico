# CV Web – Facundo Gómez

Este repositorio contiene mi **CV técnico en formato web**, desarrollado utilizando **HTML y CSS**, con un diseño moderno, claro y enfocado en perfiles IT trainee / junior.

El objetivo del proyecto es presentar de manera accesible mi perfil profesional, las tecnologías que utilizo y algunos de mis proyectos destacados.

## Descripción

El CV está implementado como una página web simple y liviana, pensada para ser visualizada desde cualquier navegador.  
Incluye información relevante sobre mi formación, habilidades técnicas y trabajos realizados, y puede utilizarse tanto como presentación personal como base para un portfolio.

## Tecnologías utilizadas

- HTML5  
- CSS3  

## Funcionalidades

- Presentación del perfil profesional  
- Listado de tecnologías y conocimientos técnicos  
- Sección de proyectos con enlaces a GitHub  
- Información de contacto y redes profesionales
